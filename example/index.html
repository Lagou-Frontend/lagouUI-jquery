    复选框

    复选框包含复选框和筛选框

    						<input type="checkbox" value="" class="checkbox" data-text="客户端" >
						<input type="checkbox" value="" class="checkbox screenbox" data-text="有面试评价">

    下拉框

    当你向一个div添加样式.dropbox或者 data-role="dropbox"两者需写一个便认为他要调用Dropbox插件，其中默认有两种方式配置参数，数据;

    第一种配置方式：通过data-setting='{}'配置，第二种在调用时配置参数

    实例





    						
						<script>
							$('.dropbox').dropbox();
						</script>
						
						<div class="dropbox" ></div>
						
						
						<div class="dropbox" data-setting='{"width":400,"height":40}'></div>
					
    						<div  data-role="dropbox" id="testDropbox"></div>
						
						$('#testDropbox').dropbox({
					    	width: 320,
					    	maxHeight : 200,
					    	placeholder : "测试测试"
					    })
					
    						
						width : '',    				默认300px
						height : '',   				默认46px
						maxHeight:'',  				默认没有
						bgcolor : "white",			默认为white可以选gray
						isOverflow : false,			默认为false
						placeholder: '请选择',		  默认为“请选择”
						data : ['2K以下','5K以下'],  默认数据，可自定义添加
						template : ''				建议为默认模板，一般不作更改，可以去源码更改

    下拉多选框

    当你向一个div添加样式.multiselect或者 data-role="multiselect"两者需写一个便认为他要调用Multiselect插件，其中默认有两种方式配置参数，数据;

    第一种配置方式：通过data-setting='{}'配置，第二种在调用时配置参数

    实例





    						
						<script>
							$('.multiselect').multiselect();
						</script>
						
						<div class="multiselect" ></div>
						
						
						<div class="multiselect" data-setting='{"width":400,"height":40}'></div>
					
    						<div  data-role="multiselect" id="testMultiselect"></div>
						
						$('#testMultiselect').multiselect({
					    	width: 320,
					    	maxHeight : 200,
					    	placeholder : "下拉测试",
					    	multiNumTips: "(最多选择三个)",
    						multiNum : 3
					    })
					
    						
						width : '',    				默认300px
						height : '',   				默认46px
						maxHeight:'',  				默认没有
						bgcolor : "white",			默认为white可以选gray
						isOverflow : false,			默认为false
						multiNum : 2,				设置允许多选个数
						placeholder: '请选择',		  默认为“请选择”
						multiNumTips :''			(最多可选择两个)
						data : ['2K以下','5K以下'],  默认数据，可自定义添加
						template : ''				建议为默认模板，一般不作更改，可以去源码更改

    下面附上其常用配置。

    实例
